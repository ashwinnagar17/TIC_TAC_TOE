# TIC_TAC_TOE
This repository contains a Tic Tac Toe game developed using Android Studio. The game offers a classic two-player experience, where players take turns marking X and O on a 3x3 grid. The objective is to get three marks in a row, column, or diagonal to win the game.

Features
Simple and intuitive user interface.
Two-player mode: Play against a friend locally.
Interactive gameplay: Tap on the grid to place your mark.
Win detection: The game automatically detects and declares the winner.
Restart option: Start a new game anytime.

Getting Started
To get started with the Tic Tac Toe game in Android Studio, follow the steps below:
Clone or download this repository to your local machine.
Open Android Studio and select "Open an Existing Project".
Navigate to the directory where you cloned/downloaded the repository and select the project.
Build and run the project on an Android emulator or a physical device.

How to Play
Launch the Tic Tac Toe game on your Android device.
The game board will be displayed as a 3x3 grid.
Two players take turns tapping on the grid to place their respective marks (X or O).
The game will automatically detect when a player has won or if the game ends in a draw.
If a player wins, a message will be displayed declaring the winner.
